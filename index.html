Default Deny, Not Default Pray

Block everything except what you explicitly allow using a human readable YAML policy file.

2

Parent-Aware Enforcement

Trust individual files or entire folders, it's your choice. When you trust a directory, all files within it and its subdirectories automatically inherit that trust. This means you can specify single programs when needed, or trust whole folder structures to avoid listing every file individually.
